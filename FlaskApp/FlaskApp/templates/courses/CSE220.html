System Fundamentals I is a course which focused on the essential concepts of
computer organization and system architecture.
Alongside being introduced to computer architecture, the course also
introduced assembly language programming.
  • Throughout this course, we learned about various topics such as
    • Number Systems
    • Memory Organization
    • MIPS Assembly & MIPS Instruction Set
    • Digital Logic & Expression Simplification
    • Multiplexers
    • Arithmetic Logic Units
    • Various Instruction Cycles
    • Single Cycle CPUs
    • Multi Cycle CPUs
    • Pipelining, Hazards, Flushing, and Instruction reorganization.
  • Each of the homework assignments consisted of an increasingly difficult MIPS programming assignment, utilizing proper register conventions, function calls, parameter passing, conditional branching, and linking.
  • As we progressed through the course, we implemented various functions using recursive techniques, stacks, 2 dimensional arrays, bit manipulation, and system calls.
  • The course also teaches students how Muxes and ALUs are used in order to create Single/Multi cycle CPUs, as well as how to implement new instructions such as "addi" into the current Single/Multie cycle CPU.
  • Finally, we ended the course by pipelining instructions to maximize effeciency, taking the necessary precautions of flushing the pipe when conflicting instructions occur and how the compiler can reorganize instructions to reduce these conflicts.
