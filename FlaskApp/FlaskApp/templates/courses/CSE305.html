Principles of Database Systems focuses on the student's ability to write SQL, convert SQL to relational agebra, and create database systems which can be used on a large scale.
  • This course covers topics which are necessary for database developers to understand and take advantage of such as
    • ER-Modeling
    • Functional Dependencies & Constraints
    • Normal Forms & Converting Schemas into 3NF or BCNF
    • Relational Algebra & Conversion into SQL
    • Triggers and Active Databases
    • Transactions & DBMS
  • Throughout the semester, the assignments consisted of a three part project.
    • The first assignment was to design an ER-Model for an E-Commerce application, which we would be implementing in the next two assignments.
    • Next, the second assignment was to implement the ER-Model we designed in the previous assignment as SQL relations with all the appropriate constraints.
    • The final assignment was to integrate the previously created SQL relations with a web application to make a fully working application which uses all the knowledge gained over the semester!
  • Overall, this course has allowed students to make relations which are efficient, by converting relations from lower normal forms into higher normal forms through lossless decomposition.
