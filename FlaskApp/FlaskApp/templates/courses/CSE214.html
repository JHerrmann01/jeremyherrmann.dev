Computer Science II is a course which focuses on data structures, sorting/searching
algorithms, recursion, and space/time complexity.
  • Some of the data structures that were discussed were
    • Stacks
    • Queues
    • Lists
    • Binary Trees
    • Heaps
    • Priority Queues
    • Balances Trees
    • Directed/Undirected Graphs
    During the semester, we focused on their use cases, as well as how to implement each of these data structures in Java.
  • When discussing recusion, we branched out into dynamic programming including the benefits of
    • Top-Down Recursion
    • Bottom-Up Recursion
  • Following recusion, we began learning about searching and sorting algorithms such as
    • Searching:
      • Hashing(Along with Clustering Problems) & Chained Hashing
      • Binary Search & Serial Search
    • Sorting:
      • Selection Sort
      • Bubble Sort
      • Insertion Sort
      • Merge Sort
      • Quick Sort
      • Heap Sort
      • Counting Sort
      • Radix Sort
      • Bucket Sort
